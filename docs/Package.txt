# Package

*ゲーム外でのアイテム入手 [#h26ac216]
ゲーム内だけでは入手できないアイテムの入手方法についてのまとめページ
アイテム入手の流れ：ゲーム外でアイテムチケット入手→ゲーム内でコード入力
[[ECOくじ>kuji]]と[[ゲーム内ショップ>ECOShop]]については各ページ参照
チケット入手方法は広義の課金によるものが多いが、イベント等で無料配布されたものもある。
関連：[[アイテムチケットの使い方>http://econline.gungho.jp/manual/howtoticket.html]]
[[チケットの有効期限にご注意ください>http://econline.gungho.jp/news/important/ticketexpirationdate.html]]
