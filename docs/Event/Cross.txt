# Event/Cross

*クロスイベント [#if27c9bc]

#contents

公式より、
 ■機械のココロ、機械のキオク。僕らはここで交差する――！
 相互に重なり合うストーリーラインを持つ、DEM専用と、DEM以外の三種族用ストーリーイベントを実装します。
 エミル世界のアクロポリスシティ、ダウンタウンにある酒屋のフィリップの家に、エミル君が大怪我をした旅人を運び込んだようです。
 詳しい話はわからないのですが、その旅人は機械の体を持っているとか……

**クロスイベントについて [#s6fcc967]
-既存三種族(エミル、タイタニア、ドミニオン)とDEMとではイベントタイトルや内容が異なります。
-このページでは既存三種族での内容を記します。[[オートマタ入手イベントについては該当ページ>Event/Automata]]をご覧下さい。
