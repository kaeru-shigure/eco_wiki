# Skill/Elementaler/Table

#author("2016-11-02T08:19:57+00:00","","")
-エレメンタラーのスキルは全体的にMOBの属性に左右されやすいため、一概にどれがいいとは言い切れません。
狩場や状況にあわせて自分で構成を考えることもしてみましょう。
-型：アクティブスキル＝''&color(red){A};''、パッシブスキル＝''&color(blue){P};''
-評価
--上位互換スキルがある場合は習得Lv近辺で評価
☆:必須クラス
◎:取得推奨
◯:あると便利
△:状況次第で役に立つけど基本役に立たない
×:不要
